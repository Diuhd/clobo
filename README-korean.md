# Clobo

(아마도) 차세대 AI 툴킷. <s>AI 버블 터지면 ㅈ되는데</s>

>이 프로젝트는 SQL이랑 디자인 철학을 제외한 모든 부분에서 바이브 코딩하지 않았습니다.
(SQL이랑 디자인 철학에 수백 시간을 쓰고 싶진 않았습니다)

현재는 Windows만 지원하며, MacOS나 리눅스는 지원할 계획이 없습니다(너무 귀찮아요).

이 프로젝트는 제 첫 Rust 프로젝트이므로, unsafe 코드나 비효율적인 부분이 있다면 신란하게 비판해 주세요.

*이 프로젝트는 아직 개발 진행 중입니다. 지금은 기능이 몇 가지뿐이지만, “곧” 더 추가될 예정입니다.*

### 로드맵

1. 어떤 앱에서든 텍스트를 하이라이트하고 AI로 활용하기

2. 클립보드 내용을 AI로 보내고 바로 응답 받기

3. 전용 파일 고정(pinning) 및 빠른 접근 기능

4. 로컬 에이전트 지원

이 프로젝트가 마음에 들거나 오늘 따라 기분이 좋으시다면 레포지토리에 별 하나 부탁드립니다!
